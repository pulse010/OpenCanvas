# The Goal
My goal making this software is to eventually create a creative suite of free open source software so that aspiring creatives dont have to pay for subcriptions that they could formerly not afford, Such as Adobe Products and Fl Studio.

OpenCanvas is designed to be:
* **Fully Professional:** Incorporating high-level tools capable of industry-standard output.
* **100% Cross-Platform:** Native support for Windows, macOS, and **LINUX** oh yeah baby...
* **Ecosystem-Minded:** Built to seamlessly integrate with future open-source creative tools in my planned suite.

I hope to create a new wave of artists that dont have to take out a loan to do what they are passionate about, and creating a enviroment of software that works wells, includes professional level features, works on Windows, MacOS, AND **LINUX** and seamlessly integrates with all the other software i plan to create in the future. 
(hint: i started this project because i cant switch to linux without my adobe software and fl studio)

# My AI Guidelines
We believe that tools should empower the creator, not replace them. To maintain the integrity, originality, and security of our codebase, we enforce the following rules regarding AI-assisted coding:

1. **Human Authorship First:** AI should only be used as a tool to assist your workflow, help solve bugs and issues, help you when you are stuck, not to generate code from scratch.
2. **Total Comprehension:** You must fully understand every line of code you submit. If code is suggested by an AI, you must be able to explain its logic, security implications, and architecture when you review your code.
3. **No Plagiarism:** Ensure that any AI-assisted code does not violate upstream copyrights or licenses.
